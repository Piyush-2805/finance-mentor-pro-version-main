# FinanceMentor Pro

A structured, AI-powered finance learning platform. Takes you from complete beginner to CFA-level professional mastery.

## What's Inside

- **Learning Path** — Foundation → Core Finance (CFA L1) → Advanced (CFA L2) → Professional Mastery (CFA L3)
- **Concept Library** — Every topic with AI explanations, formulas, worked examples, and real deals across all sizes
- **6 Calculators** — DCF (with sensitivity table), LBO Returns, Black-Scholes, Bond Analyzer, WACC Builder, Equity Models
- **Case Studies** — 15 real deals across 5 size tiers: Micro (<$100M) to Mega (>$100B)
- **Ratio Reference** — 55+ financial ratios with interpretation and real examples
- **Visual Diagrams** — Interactive LBO structure, DCF funnel, three statements map, yield curve, capital structure seesaw
- **AI Tutor** — Full chat interface powered by Claude
- **Practice Questions** — AI-generated by topic, difficulty, and type
- **Flashcards** — Spaced repetition with SM-2 algorithm

## Deployment to Vercel (5 minutes)

### Step 1 — Get a Claude API Key

1. Go to **console.anthropic.com**
2. Create a free account (phone verification required)
3. You'll receive ~$5 in free credits automatically
4. Go to **API Keys** → **Create Key**
5. Copy the key (starts with `sk-ant-api03-...`)

### Step 2 — Upload to GitHub

1. Create a free account at **github.com**
2. Click **New Repository** → name it `financementor-pro`
3. Upload all files from this folder into the repository
   - Drag and drop the entire folder contents into GitHub
   - Click **Commit changes**

### Step 3 — Deploy on Vercel

1. Go to **vercel.com** → Sign up with your GitHub account
2. Click **Add New Project**
3. Import your `financementor-pro` repository
4. Click **Deploy** — Vercel auto-detects Vite
5. Your app will be live at `financementor-pro.vercel.app` in ~60 seconds

### Step 4 — Add Your API Key in the App

1. Open your live app
2. Click **Settings** (gear icon, bottom-left)
3. Paste your Claude API key
4. Click **Save** and **Test Connection**
5. All AI content is now unlocked

## How Content Works

All educational content is AI-generated on first load and **cached in your browser** (localStorage).
- First time you open a concept: Claude generates it (~1–2 seconds, small API cost)
- Every visit after that: loads instantly from cache, zero API cost
- To regenerate: Settings → Clear Cached Content
